Ссылки в социальные сети на Jekyll (trlt)

September 30, 2017

Я видел, что эта ошибка повторяется на многих блогах. Люди не компилируют свои социальные ссылки с помощью Jekyll. Вместо этого они жестко вставляют их в HTML-код. Но, есть лучший способ, и это довольно просто.

Предпосылки

Конечно, первым условием является установка Jekyll. Для тех из вас, кто не знает, Jekyll - это простой генератор статического сайта. Он позволяет создавать веб-сайт, не беспокоясь о таких вещах, как базы данных. Второе условие - установить шрифт Awesome. Шрифт Awesome является самым популярным иконочным шрифтом и содержит сотни иконок. Пакет значков полностью открыт, поэтому вы можете использовать его в любом проекте. Чтобы включить их в свой проект, откройте каталог _includes в корневой папке вашего проекта. Затем, используя ваш любимый текстовый редактор, отредактируйте файл head.html.

Вставте эту строку в нижней части файла:

<link rel="stylesheet" href="//maxcdn.bootstrapcdn.com/font-awesome/4.3.0/css/font-awesome.min.css">

Вот и все, вы закончили с настройкой=).

Создание переменных для значков

Отредактируйте файл _config.yml в корневом каталоге. Там вы увидите две переменные, называемые twitter_username и github_username. Теперь добавьте больше переменных:

twitter_username:     dethcats
github_username:      vomk
linkedin_username:    
google_plus_username: 103459417251313685406
facebook_username:    vlad.omelchuck.5
dribbble_username:

Конечно, вам нужно будет заменить мои имена пользователей вашими именами. Если вы не используете некоторые социальные сети, которые я перечислял, просто оставьте имя пользователя пустым и не беспокойтесь об этом.

Добавление иконок

Найдите подходящее место, где вы хотите показать свои значки. Если вы хотите показать их в футере, отредактируйте файл footer.html из вашей папки _includes. Если вы хотите добавить их где-нибудь в основной части своего сайта, отредактируйте файл index.html из корневой папки.

После того, как вы нашли место, вы готовы к следующему шагу. Создайте неупорядоченный список, используя тег <ul> </ul>. Затем скопируйте приведенные ниже коды. Теперь, что эти коды будут делать, они проверют ваш файл _config.yml, который вы редактировали ранее. Если они найдут подходящее имя пользователя, они покажут значок этой социальной сети, связанный с вашим профилем.

Twitter код

{% if site.twitter_username %}
  <li>
    <a href="https://twitter.com/{{ site.twitter_username 

}}">
      <i class="fa fa-twitter"></i> Twitter
    </a>
  </li>
{% endif %}

GitHub код

{% if site.github_username %}
  <li>
    <a href="https://github.com/{{ site.github_username }}">
      <i class="fa fa-github"></i> GitHub
    </a>
  </li>
{% endif %}
