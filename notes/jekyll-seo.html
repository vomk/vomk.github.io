SEO для вашего блога на Jekyll (trlt)

September 28, 2017 • (0)

Есть несколько простых вещей, которые улучшат SEO вашего блога, вот список изменений, которые можно внести в шаблон:

1) Не следует включать название блога в тег заголовка

Чистый шаблон блога включает название сайта в каждом заголовке страницы:

<title>
{% if page.title %}{{ page.title }} - {{ site.title }}{% else %}{{ site.title }}{% endif %}
</title>

Лучше указывать название блога только на главной и на странице контактов. Я сделал вот так:

{% if page.title %}
	<title>{{ page.title }}</title>
{% else %}
	<title>{{ site.title }}</title>
{% endif %}

2) Используйте конкретное мета-описание для каждой страницы блога

<meta name="description" content="{% if page.descript %}{{ site.description }} {{ page.descript }}{% else %}{{ site.description }}{% endif %}">

И потом на каждом создаваемом MD-файле прописывайте:

descript: Улучшение SEO для вашего блога на Jekyll. Мелкие хаки для улучшения продвижения и индексации

3) Используйте meta rel = author для каждой страницы блога

Свяжите страницы с профилем google+:

<link rel="author" href="https://plus.google.com/u/0/103459417251313685406"/>

4) Создайте карту сайта для своего блога

Карта сайта предоставляет информацию поисковой системе о структуре вашего сайта и страницах, которые она должна индексировать.

Вы можете легко сгенерировать его, добавив gem jekyll-sitemap в ваш _config.xml, просто следуйте этим инструкциям.

5) Добавьте файл robots.txt

Файл robots.txt предоставляет информацию поисковикам о страницах, которые вы хотите исключить из индексации. Вы также должны указать путь к вашей карте сайта.

Добавьте аналогичный файл в корень вашего проекта:

# www.robotstxt.org/

User-agent: *
Sitemap: http://вомк.name/sitemap.xml

На этом у меня всё. А у вас есть какие-то дополнения по улучшению SEO вашего блога?
